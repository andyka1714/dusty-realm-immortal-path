- 來源畫面：使用者提供的 2026-07-19 桌機截圖（1656 × 1081）。

## 視覺比對結果

| 項目 | 結果 | 驗證重點 |
| --- | --- | --- |
| 主功能視窗 | 通過 | 外框、內層面板、按鈕、關閉控制均採用共用米紙紋理與分層紙緣陰影。 |
| HUD 與任務卡 | 通過 | 由平面深色卡片改為可辨識的紙材層級；文字對比仍符合暗色遊戲場景。 |
| 新手身份流程 | 通過 | 手機 390 × 844 無水平溢位，表單、性別選擇及確認按鈕均保有可點擊尺寸。 |
| 感靈石 | 通過 | 移除方形光框，改用依透明輪廓計算的 `drop-shadow` 呼吸光。 |
| 天命結果 | 通過 | 面板使用真實紙張材質、厚紙緣及多層落影，不再是單純古風 CSS 卡片。 |
| 野外地圖 | 通過 | runtime 改用生成式紙雕地形素材；冰、草、石、水、熔岩、虛空與仙境依地圖語意切換。 |
| 怪物 | 通過 | runtime 改用透明紙雕怪物圖；前、後、左、右分向，四足怪不再旋轉同一張側視圖。 |
| 響應式版面 | 通過 | 390 × 844 與 1656 × 1081 實機瀏覽器截圖均無水平溢位或破圖。 |

## 自動化量測

- 桌機：`horizontalOverflow = 0`、`brokenImages = 0`。
- 手機：`horizontalOverflow = 0`、`brokenImages = 0`。
- 正式包：364 個素材檔、10.41 MiB、0 個禁用檔、0 組重複檔，低於 50 MiB gate。

## 已知限制

- 目前怪物以 8 種紙雕身形原型覆蓋全部怪物資料，並依資料屬性套用方向與尺寸；不是每一個怪物都已有獨立造型。
- 地圖已全面使用生成紙雕材質，但仍沿用現有格子與語意地形資料；後續可逐張地圖補充專屬地標插畫，不影響目前碰撞與存檔格式。
