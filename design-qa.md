## 視覺比對結果

| 項目 | 結果 | 驗證重點 |
| --- | --- | --- |
| 主功能視窗 | 通過 | 外框、內層面板、按鈕、關閉控制均採用共用米紙紋理與分層紙緣陰影。 |
| HUD 與任務卡 | 通過 | 由平面深色卡片改為可辨識的紙材層級；文字對比仍符合暗色遊戲場景。 |
| 新手身份流程 | 通過 | 手機 390 × 844 無水平溢位，表單、性別選擇及確認按鈕均保有可點擊尺寸。 |
| 感靈石 | 通過 | 移除方形光框，改用依透明輪廓計算的 `drop-shadow` 呼吸光。 |
| 天命結果 | 通過 | 面板使用真實紙張材質、厚紙緣及多層落影，不再是單純古風 CSS 卡片。 |
| 野外地圖 | 通過 | 已移除逐格紋理拼貼，改用 6 張完整生成式紙雕地圖底圖；道路、水域、橋梁與地形均為連續構圖。 |
| 怪物 | 通過 | 8 種身形均具前、後、左、右分向，含移動與攻擊動作幀；四足怪不再旋轉同一張側視圖。 |
| 怪物動態 | 通過 | 移除全身呼吸縮放；移動由腿部、翅膀、身體或根系動作構成，攻擊使用獨立姿勢與紙雕技能效果。 |
| 響應式版面 | 通過 | 390 × 844 與 1656 × 1081 實機瀏覽器截圖均無水平溢位或破圖。 |
