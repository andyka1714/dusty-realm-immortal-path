## ADDED Requirements

### Requirement: 圖鑑物品頁必須依物品線分類
圖鑑物品頁必須 (MUST) 以玩家可理解的物品線分類呈現正式物品，並保留境界分段、description、品質與來源追蹤。

#### Scenario: 玩家按分類瀏覽物品
- **WHEN** 玩家打開圖鑑物品頁
- **THEN** 介面必須提供功法秘卷、裝備、丹藥、煉丹材料、煉器材料、任務物品、地區特產、貨幣代幣、符籙、陣盤、法寶器靈、突破物與其他分類
- **AND** 玩家選擇分類後，物品列表必須只顯示該分類內的物品
- **AND** 每張物品卡必須顯示主分類、原始子類、description、品質與來源追蹤
