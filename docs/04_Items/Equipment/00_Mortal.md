# 凡人期 (Mortal Realm) 裝備設計

凡人期為踏入修仙界之前的階段。此時玩家尚未轉職，且剛接觸修仙世界，裝備多為凡世間的物品。

**設計定位**：通用過渡裝備，屬性基礎，無職業偏向。

## 1. 新手通用套裝 (Novice Set)

> **平衡基準**: 針對 Boss (HP 2000 / 攻 80 / 防 42) 設計。
> **通關門檻**: 需達到 "上品 (1.4x)" 品質才能不依賴消耗品穩定通關。
