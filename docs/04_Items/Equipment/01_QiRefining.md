# 練氣期 (Qi Refining Realm) 裝備設計

練氣期是修仙者正式踏入大道的初步階段。雖然尚未正式轉職（築基期），但此時玩家可以選擇不同的裝備來強化特定的屬性，為未來的流派打下基礎。

裝備分為三大風格：**靈劍 (攻擊)**、**鍛體 (防禦)**、**五行 (法術)**。
