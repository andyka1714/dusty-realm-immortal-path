# HD 2D × 紙雕仙俠視覺規範（v0.1）

## 核心方向

採用清晰的 3/4 視角 HD 2D 角色與怪物，搭配紙張纖維、疊紙邊緣、柔和投影與墨線。地圖保留格子碰撞與腳底錨點，但畫面使用分層 raster 背景、可排序 props 與少量粒子，避免像網站卡片，也避免傳統低解析像素的素材限制。

## 共用規則

- 角色、NPC、四足怪物：前／後／側三視圖；左右由 runtime 鏡像，腳底錨點固定在碰撞格中心。
- UI、彈窗、技能、物品、裝備：同一紙張底材與 1–2 層投影；互動狀態只改墨線亮度、邊緣色與五行光暈。
- 品階不換基礎圖：沿用同一 silhouette，透過邊框、紙層數、粒子與五行色帶表現普通／良／優／極品／天工。
- 地圖至少拆成 base、walkable overlay、props、effects 四層；碰撞資料與美術圖分離。
- 手機以 390×844 為基準驗證，重要觸控目標至少 44px；桌機保留 1280×720 讀取密度。

## 交付順序

先完成玩家、守山靈虎、仙緣鎮一張代表地圖、技能圖示與裝備品階框的 pilot；通過桌機／手機 Product Design 稽核後，再批次生成其餘素材。來源素材先保留，只有通過 runtime manifest 的檔案才進正式包；未使用檔案移至可恢復 quarantine，不直接刪除。
