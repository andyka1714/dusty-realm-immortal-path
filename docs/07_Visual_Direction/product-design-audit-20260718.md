# Product Design 稽核：HD 2D 紙雕整合

證據：`output/playwright/audit-paper-cut-20260718/`

1. 故事敘事（健康）：390×844 無橫向溢出；實際地圖底圖降低亮度後成為故事舞台，文字仍清楚。
2. 姓名與性別（健康）：表單使用紙雕 panel；輸入、性別與確認按鈕具有明確層級。
3. 感靈石（健康）：已由 CSS 圓形替換成真正的紙雕靈石素材；點擊目標大於 44px。
4. 靈根結果（健康）：資訊完整且可捲動；主要 CTA 固定在內容末端，無破圖。
5. 首次進入地圖（健康）：仙緣鎮以紙雕地圖和遊戲式入境卡呈現，HUD、任務與 dock 保持可用。
6. 桌機冒險（健康）：1440×1000 地圖可視區明確；HUD 與小地圖不互相遮擋。
7. 桌機行囊（健康）：四類測試物品皆顯示紙雕圖示；視窗無網站式白卡感。
8. 桌機技能（健康）：主動與被動技能有一致圖示語言，裝備動作仍清楚。
9. 手機冒險（健康）：390×844 無水平溢出，HUD 與小地圖間距 12px。
10. 手機行囊（健康）：四欄圖示可讀，捲動與關閉按鈕可操作。

截圖只能證明可見畫面、溢出與基本互動；完整鍵盤、螢幕閱讀器與色彩對比合規仍需專門的 accessibility 測試。
