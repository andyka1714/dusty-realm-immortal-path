# 修煉機制 (Cultivation Mechanics)

## 1. 核心循環
1. **累積修為**: 透過時間自然增長 (Tick) 或手動運功。
2. **瓶頸**: 當修為達到 `MaxExp` 時停止增長。
3. **突破**: 消耗資源或通過機率檢定進入下一層。

## 2. 閉關模式 (Seclusion)
- **觸發**: 玩家主動開啟。
- **效果**: 修煉速率大幅提升 (5x)，但無法進行其他操作。
- **中斷**: 玩家手動關閉，或修為已滿自動中斷。
