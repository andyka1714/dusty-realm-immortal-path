# 修煉機制 (Cultivation Mechanics)

## 1. 核心循環
1. **時間流逝**: 全局時間恆定流逝 1秒 = 1天。
2. **累積修為**: 
   - **自然增長**: 基礎速率 (1.0x)。
   - **閉關 (Seclusion)**: 速率提升為 **5.0x**，時間照常流逝。
3. **瓶頸**: 當修為達到 `MaxExp` 時停止增長。
4. **突破**: 消耗資源或通過機率檢定進入下一層。

## 2. 閉關模式 (Seclusion)
- **觸發**: 玩家主動開啟。
- **限制**: 無法進行移動、戰鬥、煉丹等操作。
- **中斷**: 手動關閉或修為滿。

## 3. 死亡與轉生 (Death & Reincarnation)
- **判定**: 當 `當前年齡 >= 壽命上限` 時觸發。
- **流程**:
  1. 彈出死亡視窗，顯示生平總結 (存活時間、最高境界)。
  2. 數據重置 (保留部分傳承? 待定)。
  3. 點擊「轉生」按鈕重新開始新的人生。
