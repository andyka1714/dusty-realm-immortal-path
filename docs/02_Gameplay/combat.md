# 戰鬥流程 (Combat Mechanics)

## 1. 戰鬥循環 (Turn-Based)
戰鬥採用半回合制 (或即時回合制)。

1. **遇敵**: 玩家進入怪物所在的 Grid。
2. **初始化**: 生成怪物實例，鎖定玩家移動。
3. **回合結算 (Tick)**:
   - 雙方根據攻速/先手值決定行動順序 (目前簡化為同時攻擊?)。
   - 計算傷害 [公式參考](../01_Core/formulas.md)。
   - 扣除 HP。
4. **勝負判定**:
   - 怪物 HP <= 0: 勝利，獲得掉落與經驗。
   - 玩家 HP <= 0: 失敗，重傷/死亡傳送回城。
