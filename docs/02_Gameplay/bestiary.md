# 塵寰圖鑑 (Detailed Bestiary)

本圖鑑記錄了《塵寰仙途》中所有區域地圖的專屬生態系統。目前已實現 **68 張地圖與怪物的 1:1 深度匹配**，總計 **300+** 種獨特妖獸。

## 1. 生態系統規則 (Ecosystem Rules)

- **尋常 (Common)**：每張地圖包含 2-4 種特有尋常怪物。
- **精英 (Elite)**：每張地圖包含 1-2 種特有精英怪物，具備追擊能力。
- **妖王 (Boss)**：特定地圖的守關領主，控制著通往下一境界的關鍵。
- **屬性融合**：怪物屬性完全契合地圖描述（如雪山怪多為水/冰系，雷澤怪多為雷/金系）。

## 3. 戰鬥小貼士

1.  **地圖適應性**：由於怪物與地圖高度綁定，推薦玩家根據當前探索的地圖切換對應的屬性防禦或攻擊手段。
2.  **精英警戒**：精英怪物（如「地宮殺手」或「樞紐觀察者」）在地圖中具有更高的威脅，請留意其顏色標示（藍色）。
3.  **極限挑戰**：仙帝境界的「彼岸終點」怪物擁有極高的生命值與攻擊力，建議集齊法則之力後再行挑戰。
