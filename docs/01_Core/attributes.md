# 角色屬性 (Attributes)

## 1. 六維屬性定義
角色擁有六項基礎屬性，這些屬性決定了角色的核心能力成長：

- **體魄 (Physique)**: 影響 HP (氣血) 上限、物理防禦力、以及初始壽元 (100歲 ± 5)。
- **根骨 (RootBone)**: 影響 基礎攻擊力 (物理/法術)、修煉基礎速率。
- **神識 (Insight)**: 影響 MP (靈力/神識) 上限、暴擊率 (Crit Rate)。
- **悟性 (Comprehension)**: 影響 境界突破成功率、領悟功法速度 (尚未實作)。
- **福緣 (Fortune)**: 影響 掉寶率 (Drop Rate)、閃避率 (Dodge)、奇遇觸發機率 (尚未實作)。
- **魅力 (Charm)**: 影響 NPC 互動好感度、商店交易折扣 (尚未實作)。

## 2. 初始屬性生成規則 (Generation)

### 2.1 生成範圍
初始屬性值範圍為 **10 ~ 20** 點。
此數值受 **靈根資質 (Spirit Root Tier)** 影響，靈根越佳，初始屬性總和越高。

### 2.2 靈根加成公式
所有角色基礎各屬性皆為 **10 點** (基礎總和 60)。根據靈根類型給予額外點數：

| 靈根類型 (Tier) | 額外點數 | 目標總和 |
| :--- | :--- | :--- |
| **雜靈根 (Mixed)** | +10 | 70 |
| **真靈根 (True)** | +20 | 80 |
| **天靈根 (Heavenly)** | +30 | 90 |
| **變異靈根 (Variant)** | +30 | 90 |

### 2.3 分配邏輯
系統將額外的點數 (10/20/30) 隨機分配至六項屬性中。
- **限制**: 單項屬性在初始生成時，上限為 **20 點**。
- **結果**: 玩家初始單項屬性必定介於 10 (下限) 至 20 (上限) 之間。
