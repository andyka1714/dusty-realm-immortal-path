# 平衡目標與改版建議

本文件把前面幾份審計收斂成「現在該怎麼繼續修」。

---

## 1. 我們現在已經站在哪裡

這一輪之後，已經完成的支架有：

1. 修為通道關係已改正
2. 戰鬥公式已從硬門檻改成平滑減傷
3. 職業成長已經真的套用
4. 練氣技能已補上
5. 路線主題掉落已開始成立
6. 已經有自動化平衡測試

所以後面不需要再回頭重做底層，而是開始精修節奏。

---

## 2. 我建議採用的挑戰帶

### 2.1 當前境界裝備門檻

| 玩家狀態 | 應能穩定應對的內容 |
| --- | --- |
| 裸裝 / 初始裝 | 當前境界最低檔普通怪 |
| 2~3 件下品裝 | 當前境界多數普通怪 |
| 全套下品裝 | 當前境界普通怪穩定，Boss 仍是門檻 |
| 全套下品 + 少量中品 | 當前境界精英進入可挑戰區 |
| 全套中品 + 2~4 件上品 | 當前境界 Boss 可過 |
| 3 件仙品以上 | 可挑戰下一境界普通怪 / 部分精英 |

### 2.2 目前已驗證的具體例子

練氣期目前已用測試驗證：

1. 三職業全套下品能打過同路普通怪
2. 三職業全套下品打不過同路 Boss
3. 三職業全套中品 + 4 件上品能打過同路 Boss

這代表現在至少已經有一條可以繼續往上延展的標準線。

---

## 3. 我建議後面優先補的不是數字，而是密度

### 3.1 分流期

分流期現在已經開始像樣：

- 有三路主題
- 有對應職業掉落
- 有分流 Boss

這段接下來主要是細修。

### 3.2 聚合期

聚合期才是現在真正薄弱的地方：

- 普通怪數量下降
- 精英怪數量下降
- Boss 只剩 1 隻

所以接下來最需要補的是：

1. 化神以後的精英層次
2. 聚合期的 Boss 機制差異
3. 聚合期的路線主題混融，而不是直接失去主題

---

## 4. 技能系統的下一步建議

### 4.1 現在可以先停在這裡的部分

目前已經能夠：

- 施放主動技能
- 消耗 MP
- 吃技能倍率
- 走職業輸出模式

所以現在不急著再補大量新技能。

### 4.2 接下來更值得做的事

1. 把剩餘仍偏通用模板的被動，繼續升級成逐招專屬效果
2. 補 `burn / poison / bleed / shield / armorBreak`
3. 讓 Boss 對不同職業的應對方式更鮮明

---

## 5. 修為系統的下一步建議

既然你要的是沉浸式修仙體驗，那後面的進度不應該只靠：

- 自動修煉
- 手動修煉
- 閉關

而應該逐步轉成：

| 來源 | 後期建議占比 |
| --- | ---: |
| 修煉通道 | 25% ~ 35% |
| 打怪 / 精英 / Boss | 25% ~ 30% |
| 丹藥 / 洞府 / 陣法 | 20% ~ 25% |
| 奇遇 / 任務 / 首殺 | 15% ~ 20% |

這樣才能同時保有：

- 境界越高越難
- 但不是只有掛機等時間

---

## 6. 我建議的後續改版順序

### 第一優先

1. 對齊 `docs/01_Core/*` 舊文件與目前實作
2. 把聚合期怪物 / 精英 / Boss 密度補起來
3. 補更多平衡測試案例到金丹與元嬰

### 第二優先

1. 把被動技能做成逐招效果
2. 補狀態效果
3. 補職業專屬材料與詞條池

### 第三優先

1. 補丹藥、洞府、奇遇的修為乘區
2. 重新設計後期壽元與時間感
3. 補越階挑戰與特化 Boss 體驗

---

## 7. 這份文件的結論

現在這套數值已經不是鬆散原型了。

它已經有：

- 可運作的修為節奏
- 可運作的職業戰鬥
- 可運作的分流期路線掉落
- 可驗證的平衡測試

接下來最該做的，不是再盲目重拉數字，而是：

1. 補後期內容密度
2. 補技能細節
3. 補多系統成長來源
