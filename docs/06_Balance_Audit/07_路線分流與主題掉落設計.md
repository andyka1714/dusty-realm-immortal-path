# 路線分流與主題掉落設計

這份文件專門整理地圖分流、職業主題與掉落配置，避免後續又把「有沒有缺資料」和「資料主題是否正確」混在一起。

- 凡人：三路起步
- 練氣：三宗分流
- 築基：三路完整
- 金丹：三路完整
- 元嬰：三路完整

## 2. 三條路線的主題

### 北路

- 對應職業：劍修
- 主題：劍意、寒鋒、單體爆發、破防
- 掉落偏向：
  - 劍
  - 劍穗 / 劍匣 / 護心鏡
  - 暴擊、暴傷、速度、攻擊

### 西路

- 對應職業：體修
- 主題：淬體、獸血、站樁、續戰
- 掉落偏向：
  - 拳套
  - 盾
  - 重甲
  - 體魄、氣血、防禦、減傷、回血

### 東路

- 對應職業：法修
- 主題：靈潮、幻術、湖澤、法力循環
- 掉落偏向：
  - 法杖
  - 靈珠 / 法鏡 / 法冠
  - 法傷、神識、MP、抗性、速度

---

## 4. 為什麼普通怪不需要完全鎖死

普通怪比較適合做：

- 當前路線主題掉落偏多
- 非本路線裝也能低機率掉

理由：

1. 玩家不會因為早期選路而完全鎖死
2. 仍能保留路線辨識度
3. 精英 / Boss 才能形成真正的職業養成節奏

所以目前比較合理的分工是：

- 普通怪：偏向主題，但保留混池空間
- 精英怪：主題更明確
- Boss：主題最明確，並兼顧突破素材

---

## 5. 後續建議

如果後面還要往沉浸式修仙體驗再推一步，我建議：

1. 每條路線再加 1~2 個專屬詞條池
2. Boss 掉落再加一層路線專屬材料
3. 聚合期改成「主題混融」，不要直接完全失去路線辨識
