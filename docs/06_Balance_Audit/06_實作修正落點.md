## 6. 裝備資料

修改位置：

- `data/items/equipment/*.ts`
- `utils/dropSystem.ts`

重新盤點後的結論：

- 各大境界裝備其實都有，不是缺整段
- 真正要改的是：
  - 掉落主題
  - 品階門檻
  - 職業對裝備屬性的轉換效率

如果後續要補平衡：

1. 先調 `equipment/*.ts` 基礎值
2. 再調 `dropSystem.ts` 的品質倍率與詞條
3. 先看 `data/items/equipment/audit.ts` 的總值表與詞條權重是否仍成立
4. 最後用 `battleSystem.test.ts` 驗證

若要調整三職業在不同大境界的目標擊殺時間，現在優先改：

- `data/progression/balanceAudit.ts -> REALM_TTK_TARGETS`

而不是直接先改文件。

- `data/items/equipment/equipmentAudit.test.ts`
