- 已完成的 `meta progression / workshop / pixel terrain / sect midgame / base spec truth` 相關 OpenSpec changes

到目前為止，repo 內已正式落地的基線，不再只是 battle core：

1. `Adventure` 已正式承接地圖內即時戰鬥、怪物分工、投射物、危險區與 live-combat HUD。
2. 後段高境界內容密度、路線主題掉落與 `仙帝` 終盤門檻已做完一輪收口。
3. `Soul Save / Lifetime Stats / Merit / Reincarnation Hall / 進階 perk / heirloom planner / 主動坐化` 已進入正式輪迴流程。
4. `Workshop` 不再只有聚靈陣；第一批 `煉丹 / 煉器` 與對應材料循環已是正式功能。
5. `Encounter` 已不再只是純 flavor log；現在已有 `pending encounter -> choice -> reward -> resolved` 的正式 flow，且高境界 multiplier 也已綁到真實 event ids。
   另外，pending encounter modal 已把 `categoryLabel / routeLabel` 與 choice cue tags 露出來，玩家在下選項前能先看懂風險、收益、成本與路線歸屬。
6. 三宗 `築基 -> 金丹` 的中期宗門任務與承接 NPC 已正式落地。
7. `AdventureStage` 的 terrain / background 已完成像素化整合，但玩家、怪物、NPC 仍維持現有文字 token 呈現。
8. OpenSpec base specs、persisted state migration 與目前正式 UI / loop truth 已完成一輪回寫。

這代表目前最缺的，不再是：

- 再補一次輪迴基礎
- 再補一次百業第一批入口
- 再補一次 base spec / migration truth
- 再把角色 / 怪物整包改成像素 sprite

真正還薄的缺口，已經收斂成以下三類：

1. `事件 / 奇遇` 與 `宗門 / 世界內容` 都已完成第一輪深化，但後續仍可繼續補更多內容模板。
2. `Workshop` 已有第一批 loop，但高階 recipe、專精與品質分化仍未開始。
3. 像素風目前只正式用在 terrain/background，後續若要續推，應是 production polishing，不是全專案換皮。

## 3. 重新排序後的核心判斷

### 第一判斷：下一條最該補的是高階百業，不是新基礎建設

目前 repo 已經有：

- 輪迴正式流程
- 第一批百業 loop
- base spec / persistence truth
- 宗門中期線
- encounter selector / UI cue
- 三宗 `元嬰+` 後段任務與 world milestone encounter

因此下一條主線再去補「基礎存在與否」的問題，收益已經很低。

相反地，`Workshop` 現在正卡在一個很典型的中間狀態：

- 第一批 recipe loop 已存在
- 高境界 multiplier 已有 audit 對照
- route-specific encounter / world reward source 已開始提供材料語意
- 但高階 recipe、品質分化與專精 sink 尚未開始

這代表下一輪最該做的，不是再開一套新系統，而是把既有百業 loop 推進成後期 build 可以追求的高階支撐。

### 第二判斷：宗門 / 世界後段已完成第一輪，後續應反向供料給 Workshop

宗門後段與世界里程碑原本是大缺口，但現在已由 `expand-sect-world-late-progression` 補上第一輪：

- 三宗 `task_04` 已接到 `元嬰`
- 宗門 hub NPC 已能提供後段任務
- `化神` 三界戰場里程碑 encounter 已能依 profession / completed quest gating 解鎖

因此下一步不該再停留在宗門任務數量，而是讓這些 route-specific world reward 成為高階百業的材料來源與專精依據。

### 第三判斷：百業高階深化應承接新事件材料，而不是脫鉤進行

高階百業仍然值得做，但如果不先補更強的 encounter / world reward source：

- 高階 recipe 會缺乏自然的材料來源
- 專精與品質分化容易再次變成純數值表

因此百業高階深化的正確位置，現在已經從「等待 encounter / world source」變成「下一條主線」。

### 第四判斷：像素風目前只該做地圖 production polish

目前 terrain/background 像素化已完成，而且玩家已明確要求：

- 玩家 / 怪物 / NPC 維持現有文字 token
- 只調整地圖背景與場景語言

所以像素風主線現在應被視為 production polish track：

- 補 tile / biome / palette / landmark consistency
- 不動 actor representation
- 不和 encounter / sect / workshop 的系統主線搶第一順位

## 5. 建議執行方式

目前最合理的下一步，不是再開一條泛用大主線，而是把 Workshop 從第一批 loop 推進到高階深度。

建議順序：

1. 先開 `high-tier workshop depth`
2. 把高階丹方 / 器方、品質分化、專精與材料 sink 接上
3. 再視需要補更細的 world / sect reward source
4. 像素地圖 production polish 則作為視覺產線承接其後

換句話說，現在的第一順位不再是：

- `輪迴`
- `Workshop foundation`
- `Spec / Persistence`
- `像素風整包換皮`

而是：

- **把已經存在的 Workshop loop 推進成高階 build 支撐**

## 6. 結論

依照目前 live code 與文件真相，建議優先級如下：

1. `high-tier workshop depth`
2. `像素地圖 production polish`
3. 後續再看更細的世界 / 宗門支線補強

目前已依此判斷開出的下一條 OpenSpec change 是：

- `update-high-tier-workshop-depth`

這條線的特點是：

- 不需要重做 persistence schema
- 可以直接利用已經存在的 `Workshop` 第一批 loop 與 route-specific 材料來源
- 做完之後，高階百業與後續 world / sect content 都會更容易接上
