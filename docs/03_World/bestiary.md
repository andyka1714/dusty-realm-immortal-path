# 怪物圖鑑 (Bestiary)

## 1. 怪物數據結構
怪物 (Enemy) 包含以下核心屬性：
- **ID**: 唯一標識符 (如 `wind_wolf`).
- **Rank**: 階級 (Common, Elite, Boss).
- **Element**: 五行屬性 (用於生剋計算).
- **Drops**: 掉落列表 (引用 Item ID).
