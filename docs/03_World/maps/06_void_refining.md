# 06. 煉虛期 (Void Refining)

本區域包含地圖 ID 52-54，突破化神極限，進入虛空修煉。

## 52. 虛無裂隙
- **Realm**: 煉虛
- **Size**: 150x150
- **Position**: [0, 9]
- **Theme**: North
- **Description**: 萬物歸寂，裂縫橫生。物質在此被剝離了形態，只剩下最本源的虛無。
- **Connections**:
    - South -> 51 (九天劫海)
    - North -> 53 (元神試煉場)

## 53. 元神試煉場
- **Realm**: 煉虛
- **Size**: 150x150
- **Position**: [0, 10]
- **Theme**: North
- **Description**: 心魔叢生，幻象萬千。這裡沒有真實的敵人，唯一的對手只有你自己的元神。
- **Connections**:
    - South -> 52 (虛無裂隙)
    - North -> 54 (太虛幻境)

## 54. 太虛幻境
- **Realm**: 煉虛
- **Size**: 150x150
- **Position**: [0, 11]
- **Theme**: North
- **Description**: 真假難辨，若即若離。在太虛之中，真實不過是一瞬的執念，虛幻才是永恆。
- **Connections**:
    - South -> 53 (元神試煉場)
    - North -> 55 (空間崩壞裂縫)
- **Boss**: 太虛夢魘 (煉虛圓滿 / 虛空) @ [75, 75]
