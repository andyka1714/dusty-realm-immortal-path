# 03. 金丹期 (Golden Core)

本區域包含地圖 ID 35-42，探索更深層的屬性領域。

## 35. 迷霧幻海
- **Realm**: 金丹
- **Size**: 150x150
- **Position**: [0, 6]
- **Theme**: North
- **Description**: 波光瀲灩，雲遮霧繞。天地一色間，分不清是海市蜃樓，還是真實的人間仙境。
- **Connections**:
    - North -> 22 (落雷懸崖)
    - West -> 36 (千幻石林)

## 36. 千幻石林
- **Realm**: 金丹
- **Size**: 150x150
- **Position**: [-1, 6]
- **Theme**: North
- **Description**: 石柱擎天，陣紋密布。每一根石柱都演化著一重幻境，道心不堅者必將永墮沉淪。
- **Connections**:
    - East -> 35 (迷霧幻海)
    - South -> 43 (荒古遺地)
    - South -> 37 (幻夢神海)

## 37. 幻夢神海
- **Realm**: 金丹
- **Size**: 150x150
- **Position**: [0, 7]
- **Theme**: North
- **Description**: 彼岸花開，如夢似幻。這是金丹修士的問心之地，唯有斬斷恐懼，方能得見真我。
- **Connections**:
    - NorthWest -> 36 (千幻石林)
    - South -> 43 (荒古遺地)
- **Boss**: 蜃樓主 (金丹圓滿 / 幻/水) @ [75, 30]

## 38. 腐蝕灘塗
- **Realm**: 金丹
- **Size**: 150x150
- **Position**: [0, -6]
- **Theme**: South
- **Description**: 黑水拍岸，酸霧蝕骨。每一朵浪花都帶著銷魂蝕骨的劇毒，生機在此斷絕。
- **Connections**:
    - North -> 26 (赤紅裂谷)
    - West -> 39 (沉沒古城)

- **Connections**:
