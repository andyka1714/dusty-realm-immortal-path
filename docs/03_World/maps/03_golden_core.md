# 03. 金丹期 (Golden Core)

金丹期地圖進一步強化了屬性與環境的極端性。

## 北方：劍修路線 (Sky/Air)
*連接自：落雷懸崖 [0, 10]*

### 60. 罡風層
- **Realm**: 金丹
- **Position**: [0, 11]
- **Theme**: North - Wind
- **Description**: 脫離了地表，進入高空。猛烈的罡風如同無數利刃切割著身體。
- **Connections**:
    - South -> 32 (落雷懸崖 - 築基)
    - North -> 61 (懸空浮島)

### 61. 懸空浮島
- **Realm**: 金丹
- **Position**: [0, 12]
- **Theme**: North - Floating Island
- **Description**: 違背重力法則漂浮的島嶼群，據說是上古戰場的碎片。
- **Connections**:
    - South -> 60 (罡風層)
    - North -> 62 (通天鎖鏈)

### 62. 通天鎖鏈
- **Realm**: 金丹
- **Position**: [0, 13]
- **Theme**: North - Chains
- **Description**: 巨大的鐵索橫亙天際，連接著更高的領域。行走其上，稍有不慎便是萬丈深淵。
- **Connections**:
    - South -> 61 (懸空浮島)
    - North -> 90 (萬載玄冰窟 - 元嬰)

---

## 西方：體修路線 (Gravity/Lava)
*連接自：烈焰荒原 [-10, 10]*

### 70. 重力黑山
- **Realm**: 金丹
- **Position**: [-10, 11]
- **Theme**: West - Gravity Mountain
- **Description**:這裡的重力是外界的十倍。每一步都要消耗巨大的體力，是錘鍊體魄的聖地。
- **Connections**:
    - East -> 42 (烈焰荒原 - 築基)
    - West -> 71 (熔岩煉獄)

### 71. 熔岩煉獄
- **Realm**: 金丹
- **Position**: [-10, 12]
- **Theme**: West - Lava
- **Description**: 深入地底核心，四周盡是流淌的岩漿。高溫足以融化金鐵。
- **Connections**:
    - East -> 70 (重力黑山)
    - West -> 72 (萬毒坑)

### 72. 萬毒坑
- **Realm**: 金丹
- **Position**: [-10, 13]
- **Theme**: West - Poison Pit
- **Description**: 匯聚天下奇毒，雖極度危險，但也能練就萬毒不侵之體。
- **Connections**:
    - East -> 71 (熔岩煉獄)
    - West -> 100 (盤古脊 - 元嬰)

---

## 東方：法修路線 (Ocean/Mystery)
*連接自：寂滅雷澤 [10, 10]*

### 80. 風暴洋
- **Realm**: 金丹
- **Position**: [10, 11]
- **Theme**: East - Ocean
- **Description**: 狂風巨浪從未停歇，海中巨獸在此興風作海。
- **Connections**:
    - West -> 52 (寂滅雷澤 - 築基)
    - East -> 81 (蓬萊仙島)

### 81. 蓬萊仙島
- **Realm**: 金丹
- **Position**: [10, 12]
- **Theme**: East - Island
- **Description**: 傳說中的仙島，靈藥遍地，但也伴隨著強大的守護靈獸。
- **Connections**:
    - West -> 80 (風暴洋)
    - East -> 82 (幻海古境)

### 82. 幻海古境
- **Realm**: 金丹
- **Position**: [10, 13]
- **Theme**: East - Illusion
- **Description**: 鏡花水月，真假難辨。這裡考驗的不僅是修為，更是道心。
- **Connections**:
    - West -> 81 (蓬萊仙島)
    - East -> 110 (靈魂之海 - 元嬰)
