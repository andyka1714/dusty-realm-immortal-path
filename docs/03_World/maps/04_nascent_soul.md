# 04. 元嬰期 (Nascent Soul)

本區域包含地圖 ID 43-48，從「荒古遺地」到「不朽帝尊陵」，探索元嬰極限。

## 43. 荒古遺地
- **Realm**: 元嬰
- **Size**: 120x120
- **Position**: [0, 8]
- **Theme**: North
- **Description**: 天地洪荒，歲月悠悠。漂浮的陸地碎片記錄著太古的戰火，虛空亂流在此交匯。
- **Connections**:
    - North -> 37 (幻夢神海)
    - East -> 44 (葬劍古塚)
    - West -> 49 (碎星浮空島)

## 44. 葬劍古塚
- **Realm**: 元嬰
- **Size**: 120x120
- **Position**: [1, 8]
- **Theme**: North
- **Description**: 劍氣縱橫，萬劍歸宗。無數斷劍插於荒塚之上，悲鳴聲中似乎在等待新的主人。
- **Connections**:
    - West -> 43 (荒古遺地)
    - South -> 52 (虛無裂隙)
- **Boss**: 無名劍修 (元嬰大圓滿 / 物理/劍) @ [60, 30]

## 45. 萬法枯竭荒漠
- **Realm**: 元嬰
- **Size**: 120x120
- **Position**: [0, -8]
- **Theme**: South
- **Description**: 靈氣禁絕，萬法皆空。這是一片被天道遺棄的荒漠，修仙者的修為在此將受到嚴酷考驗。
- **Connections**:
    - North -> 39 (沉沒古城)
    - NorthEast -> 40 (幽冥地宮)
    - East -> 46 (流沙死域)

## 46. 流沙死域
- **Realm**: 元嬰
- **Size**: 120x120
- **Position**: [1, -8]
- **Theme**: South
- **Description**: 黃沙漫漫，吞噬萬物。流動的沙丘如同活物，隨時準備將過客拖入無底深淵。
- **Connections**:
    - West -> 45 (萬法枯竭荒漠)
    - South -> 47 (鎮魔擎天塔)
