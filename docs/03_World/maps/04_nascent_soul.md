# 04. 元嬰期 (Nascent Soul)

元嬰期是分歧路線的最後階段，即將邁向合流。

## 北方：劍修路線 (Ice/Space)
*連接自：通天鎖鏈 [0, 13]*

### 90. 萬載玄冰窟
- **Realm**: 元嬰
- **Position**: [0, 14]
- **Theme**: North - Ice Cave
- **Description**: 寒氣已化為實質，連靈魂都能凍結。唯有元嬰之火能在這裡勉強燃燒。
- **Connections**:
    - South -> 62 (通天鎖鏈 - 金丹)
    - North -> 91 (極光天境)

### 91. 極光天境
- **Realm**: 元嬰
- **Position**: [0, 15]
- **Theme**: North - Aurora
- **Description**: 美輪美奐的極光下，隱藏著撕裂空間的恐怖力量。
- **Connections**:
    - South -> 90 (萬載玄冰窟)
    - North -> 92 (碎星劍垣)

### 92. 碎星劍垣
- **Realm**: 元嬰
- **Position**: [0, 16]
- **Theme**: North - Star Ruins
- **Description**: 傳說上古劍仙一劍斬落星辰與此。殘留的劍意歷經萬載不滅。
- **Connections**:
    - South -> 91 (極光天境)
    - North -> 120 (三界戰場 - 化神匯聚)

## 西方：體修路線 (Blood/Ancient)
*連接自：萬毒坑 [-10, 13]*

### 100. 盤古脊
- **Realm**: 元嬰
- **Position**: [-10, 14]
- **Theme**: West - Bone Mountain
- **Description**: 巨大的山脈宛如巨人的脊骨，散發著蒼茫的威壓。
- **Connections**:
    - East -> 72 (萬毒坑 - 金丹)
    - West -> 101 (龍血池)

### 101. 龍血池
- **Realm**: 元嬰
- **Position**: [-10, 15]
- **Theme**: West - Blood Pool
- **Description**: 沸騰的血池，蘊含著真龍之力。沐浴其中可脫胎換骨。
- **Connections**:
    - East -> 100 (盤古脊)
    - West -> 102 (蠻荒祖廟)

### 102. 蠻荒祖廟
- **Realm**: 元嬰
- **Position**: [-10, 16]
- **Theme**: West - Temple
- **Description**: 供奉著遠古巫神的神廟，記載著肉身成聖的秘密。
- **Connections**:
    - East -> 101 (龍血池)
    - North -> 120 (三界戰場 - 化神匯聚) - *Connection bridges gap to center*

## 東方：法修路線 (Soul/Void)
*連接自：幻海古境 [10, 13]*

### 110. 靈魂之海
- **Realm**: 元嬰
- **Position**: [10, 14]
- **Theme**: East - Soul
- **Description**: 沒有物質的海水，只有純粹的精神能量。稍有不慎便會魂飛魄散。
- **Connections**:
    - West -> 82 (幻海古境 - 金丹)
    - East -> 111 (幽冥鬼域)

### 111. 幽冥鬼域
- **Realm**: 元嬰
- **Position**: [10, 15]
- **Theme**: East - Ghost
- **Description**: 生與死的交界，鬼物橫行。法修可在此參悟輪迴之道。
- **Connections**:
    - West -> 110 (靈魂之海)
    - East -> 112 (虛空裂隙)

### 112. 虛空裂隙
- **Realm**: 元嬰
- **Position**: [10, 16]
- **Theme**: East - Void Rift
- **Description**: 空間極不穩定，常有域外天魔入侵。
- **Connections**:
    - West -> 111 (幽冥鬼域)
    - North -> 120 (三界戰場 - 化神匯聚) - *Connection bridges gap to center*
