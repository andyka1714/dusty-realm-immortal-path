# 09. 渡劫期 (Tribulation)

## 160. 劫雲荒原 (Tribulation Wasteland)
- **Realm**: 渡劫
- **Position**: [0, 25]
- **Theme**: Dark
- **Description**: 烏雲壓頂，雷聲轟鳴。空氣中充滿了毀滅的氣息，這裡是迎接天劫的最後準備地。
- **Connections**:
    - South -> 151 (登仙天梯 - 大乘)
    - North -> 161 (九天雷池)

## 161. 九天雷池 (Nine Heavens Thunder Pool)
- **Realm**: 渡劫
- **Position**: [0, 26]
- **Theme**: Thunder
- **Description**: 天劫的核心，雷霆化作池水。只有沐浴雷劫不死，方能脫胎換骨，成就仙身。
- **Connections**:
    - South -> 160 (劫雲荒原)
    - North -> 170 (接引仙殿 - 仙人)
