# 05. 化神期 (Spirit Severing)

本區域包含地圖 ID 49-51，挑戰天地規則，準備踏入虛空。

## 49. 碎星浮空島
- **Realm**: 化神
- **Size**: 120x120
- **Position**: [-1, 8]
- **Theme**: North
- **Description**: 星河破碎，島嶼懸空。這裡是離天最近的地方，舉手可摘星辰，低頭可見蒼生。
- **Connections**:
    - East -> 43 (荒古遺地)

## 50. 乾坤倒置界
- **Realm**: 化神
- **Size**: 120x120
- **Position**: [-7, -3]
- **Theme**: West
- **Description**: 陰陽逆亂，天地倒懸。重力在此失去了意義，每一步都在挑戰常理的認知。
- **Connections**:
    - East -> 42 (萬妖石窟)
    - West -> 51 (九天劫海)

## 51. 九天劫海
- **Realm**: 化神
- **Size**: 120x120
- **Position**: [-8, -4]
- **Theme**: West
- **Description**: 雷漿化海，咆哮不息。這是天道對化神修士最後的考驗，渡過即為仙途。
- **Connections**:
    - East -> 50 (乾坤倒置界)
    - North -> 52 (虛無裂隙)
- **Boss**: 霆祖 (化神圓滿 / 雷/罰) @ [30, 60]
