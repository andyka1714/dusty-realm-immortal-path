# 遊戲流程文檔 (Game Flow)

## 1. 初始劇情 (Intro Sequence)

### 1.1 序幕 (Prologue)
- **形式**：打字機效果 (Typewriter Effect)
- **內容**：
  > 大荒曆一〇二六年，春。
  > 你今年剛滿十歲，出生在青雲山腳下的一個小村莊...
  > (青雲宗下山選拔弟子，仙人冷漠點名)

### 1.2 身份確認 (Identity)
- **輸入**：
  - **姓名**：玩家自定義 (不可為空)
  - **性別**：男兒身 (Male) / 女兒身 (Female)
- **介面**：輸入框 + 性別切換按鈕 (藍色/粉色高亮)

### 1.3 靈根覺醒 (Awakening)
- **互動**：點擊感靈石 (Stone)
- **特效**：
  - 感靈石震動 (`isShaking`)
  - 粒子旋轉 (`Sparkles`)
  - 螢幕閃白 (Flash)
- **後台邏輯**：
  - 呼叫 `generateInitialStats()` 生成六維屬性 (10-30)。
  - 呼叫 `generateSpiritRoot()` 根據權重抽取靈根 ID。
    - 雜靈根 (70%)
    - 真靈根 (20%)
    - 天靈根 (8%)
    - 變異靈根 (2%)

### 1.4 天命展示 (Destiny)
- **介面佈局**：
  - **標題**：顯示靈根名稱 (如「金系天靈根」)。
  - **對話**：顯示接引仙人的評價 (`SpiritRootDef.comment`)。
  - **天命標籤**：顯示特殊體質 (如「庚金之體」)。
  - **屬性面板 (左)**：顯示六維屬性 (體魄、根骨等) 及加成。
  - **資質圖表 (右)**：顯示修煉效率、戰鬥能力、機緣氣運的百分比條。
- **進入遊戲**：點擊「逆天而行，開啟仙途」按鈕，初始化 Redux State 並跳轉至 Dashboard。

## 2. 主頁面 (Dashboard / 道始中心)

### 2.1 佈局
- **頂部**：
  - **玩家資訊**：頭像(文字A/B)、姓名、性別、境界(凡人/練氣)、層數。
  - **修煉進度條**：顯示當前經驗/上限，以及動態速率 (+13.7/秒)。
- **中部 (Actions)**：
  - **運功 (Manual)**：點擊手動獲得少量修為 (1.0x)。
  - **閉關 (Seclusion)**：切換閉關狀態 (5.0x 速率，但禁用歷練/打工)。
  - **境界突破 (Breakthrough)**：當進度條滿時解鎖，點擊嘗試突破。
- **右側 (Stats)**：
  - **HP/MP**：氣血與靈力數值。
  - **靈根/屬性**：顯示詳細屬性列表 (體魄、根骨、神識、悟性、福緣、魅力)。
- **底部 (Log)**：
  - **修煉日誌**：顯示最近的遊戲事件 (修煉成功、獲得物品、時間流逝)。

## 3. 修仙歷練 (Adventure)
> 在側邊欄點擊「修仙歷練」進入 (閉關時禁用)。

### 3.1 地圖選擇
- 顯示地圖列表 (Section 1: 凡人區, Section 2: 練氣區...)。
- 每個地圖顯示名稱、建議境界、描述。

### 3.2 探索介面
- **網格地圖**：黑色背景網格，綠點代表玩家，紅點代表敵人。
- **迷霧機制**：未探索區域為黑色，已探索顯示網格。
- **小地圖 (Minimap)**：右上角顯示全圖縮略，點擊可快速查看全貌。
- **控制盤**：右下角方向鍵 (上下左右) 控制移動。

## 4. 行囊空間 (Inventory)
- **裝備欄**：頂部顯示 武器/防具/飾品 欄位。
- **物品網格**：下方顯示背包物品，點擊可查看詳情或裝備/使用。

## 5. 洞府百業 (Workshop)
- **聚靈陣**：升級可提升基礎修煉速率。
- **煉丹房**：消耗草藥製作丹藥。
- **煉器室**：消耗礦石製作裝備。
