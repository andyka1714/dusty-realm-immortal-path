# 職業系統 (Professions)

本遊戲設有三大修行流派，玩家在築基期需選擇其一，決定未來的成長方向與戰鬥風格。

## 1. 劍修 (Sword Cultivator)

*   **定位**: 極致輸出，一擊必殺。
*   **核心屬性**: 攻擊力 (Attack)、暴擊率 (Crit Rate)。
*   **專屬武器**: **劍 (Sword)**。
*   **專屬被動**: 【人劍合一】
    *   攻擊時無視敵方 **15%** 防禦力。
    *   暴擊傷害增加 **50%**。

## 2. 體修 (Body Cultivator)

*   **定位**: 肉盾坦克，生存恢復。
*   **核心屬性**: 生命值 (HP)、防禦力 (Defense)。
*   **專屬武器**: **拳套 (Gauntlet)**。
*   **專屬被動**: 【金剛不壞】
    *   受到最終傷害減少 **20%**。
    *   每回合行動時，自動恢復 **2%** 最大生命值。

## 3. 法修 (Mage Cultivator)

*   **定位**: 法術炮台，資源循環。
*   **核心屬性**: 靈力值 (MP)、命中率 (Hit Rate)。
*   **專屬武器**: **法杖 (Staff)**。
*   **專屬被動**: 【天人感應】
    *   技能/功法傷害加成 **25%**。
    *   釋放技能後有 **20%** 機率不消耗靈力。
