# 介面與流程 (UI Flow)

## 1. 遊戲初始化流程 (Initialization Flow)
1. **序幕動畫 (Prologue)**: 自動播放打字機文本 -> 1500ms 延遲 -> 轉場。
2. **身份確認 (Identity)**:
   - 輸入姓名 (Input)
   - 選擇性別 (Toggle: Male/Female) -> 影響頭像與稱謂。
3. **靈根抽取 (Gacha)**:
   - 點擊石頭 -> 震動動畫 -> 閃光 -> 展示結果。
4. **天命展示 (Destiny)**:
   - 顯示靈根類型、六維屬性表、資質雷達圖。
   - 點擊「開啟仙途」 -> 初始化 Redux 狀態 -> 進入 Dashboard。

## 2. 主頁面 (Dashboard)
位於側邊欄的第一個 Tab「道始中心」。

### 2.1 資訊區 (Header)
- **左側**：頭像 (根據性別顯示 A/B)、Realm Badge (境界標籤)。
- **中間**：修煉進度條 (Progress Bar)。
  - 顯示：`當前修為 / 上限 (+速率/秒)`。
  - 動畫：進度條應平滑增長。

### 2.2 操作區 (Actions)
- **運功 (Manual)**: 點擊獲得 1 秒修為。
- **閉關 (Seclusion)**: Toogle 開關。
  - 開啟時：修煉速率 x5，但無法進入「歷練」或「洞府」。
  - 關閉時：恢復正常速率。
- **突破 (Breakthrough)**: 僅在進度條滿 (100%) 時高亮顯示。

### 2.3 屬性區 (Stats Panel)
- **HP/MP**: 雙色數值條 (紅/藍)。
- **詳細屬性**: 網格列出 6 大屬性，若有裝備加成需顯示綠色 `+XX`。

## 3. 歷練介面 (Adventure UI)
- **地圖列表**: 根據境界分區 (Block)，鎖定未達標區域。
- **探索視圖**:
  - **Grid**: 視窗中心為玩家，周圍繪製網格。
  - **Fog of War**: 僅顯示視野範圍 (預設 3x3 或 5x5) 內的格子。
  - **Minimap**: 右上角懸浮窗，顯示已探索區域。
  - **Controls**: 右下角虛擬方向鍵 (Mobile 優化)。

## 4. 行囊介面 (Inventory UI)
- **裝備槽**: 頭部、身體、武器、飾品。
- **物品欄**: 網格佈局 (Grid)，滑鼠懸停顯示 Tooltip (包含物品描述、類型、售價)。

## 5. 洞府介面 (Workshop UI)
- **Tab 切換**: 聚靈陣 / 煉丹 / 煉器。
- **配方列表**: 點擊配方自動填入材料，若材料不足則顯示紅色。
