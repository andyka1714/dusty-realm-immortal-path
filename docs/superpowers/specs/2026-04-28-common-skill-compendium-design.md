# 通用功法與功法圖鑑卡片設計

## 背景

`功法神通` 目前已能按職業與境界分組，但卡片視覺和 `裝備法寶` 不一致，而且 `通用` 分頁沒有正式功法。這會讓圖鑑看起來像未完成資料，而不是玩家可以理解的功法來源索引。

## 目標

1. 讓功法卡片使用和裝備卡片相同的資訊節奏：名稱、類型 badge、分類列、描述、來源追蹤。
2. 補上正式通用功法，使 `通用` 分頁不再是空狀態。
3. 通用主動功法可被任一職業裝備與戰鬥選用；通用被動功法可照既有學習與圖鑑流程顯示。
4. 不改圖鑑 tab 結構，`功法神通` 仍是獨立於 `萬物圖鑑` 的主項目。

## 通用功法規劃

通用功法定位為所有路線都能使用的基礎輔助與保命手段，不取代劍修、體修、法修的職業核心輸出。第一批採 8 本正式核心功法，跨越前中後期：

- 練氣：`歸元吐納`，被動，藏經閣入門心法。
- 築基：`斂息步`，主動，位移/脫戰風格的通用術式。
- 金丹：`凝神訣`，被動，穩定神識與命中。
- 元嬰：`護身靈罩`，主動，短時間自保。
- 化神：`破妄靈目`，被動，偵破幻術與提高暴擊抗性。
- 煉虛：`縮地成寸`，主動，快速接敵或撤離。
- 合體：`萬象歸一訣`，被動，讓五行與肉身運轉更均衡。
- 大乘：`太虛一氣`，主動，後期通用爆發但低於職業終局殺招。

## 介面設計

功法卡片改為和裝備卡片同樣的 `bg-stone-800 / border-stone-700 / hover:border-amber-500/30` 卡片。卡片內容：

- 右上顯示 `主動` 或 `被動` badge。
- 第二列顯示 `功法` 分類 badge、職業/通用標籤與境界。
- 描述使用固定兩行高度，避免卡片高度過度跳動。
- 下方顯示 `來源追蹤`，沿用現有來源 chips。

## Persisted State

新增通用功法資料會新增 manual item id 與 skill id。這不改 LocalStorage schema，但 hydrate / reducer / battle selection 必須允許 `ProfessionType.None` 的主動功法作為合法裝備功法。既有存檔缺少這些 skill 不需要 migration。
